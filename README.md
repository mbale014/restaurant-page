## About 
A modern, responsive restaurant website featuring a clean hero section, a menu page, customer testimonials, and an easy-to-read layout. Built with Html, Css, and Javascript.
Design inspired by warm colors, minimal style, and simple UX principles.

## Pages included

* **Home page** : hero, menu preview, about, testimonials, contact
* **Menu page** : featured dishes, sides, prices
* **About page** : restaurant storytelling
